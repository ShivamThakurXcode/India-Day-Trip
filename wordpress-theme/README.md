# India Day Trip WordPress Theme

This WordPress theme replicates the design and functionality of the India Day Trip PHP website.

## Installation

1. Upload the `wordpress-theme` folder to your WordPress `wp-content/themes/` directory.
2. Activate the theme in WordPress admin under Appearance > Themes.
3. Upon activation, the theme will automatically create ALL pages from your PHP site (A to Z), including:
   - Home (index.php)
   - About (page-about.php)
   - Contact (page-contact.php)
   - Gallery (page-gallery.php)
   - Blog
   - Tour pages
   - Same Day Tours
   - Taj Mahal Tours
   - Golden Triangle Tours
   - Rajasthan Tour Packages
   - To Book
   - Privacy Policy
   - Terms & Conditions
   - All individual tour pages (Taj Mahal Sunrise, Golden Triangle 4N5D, etc.)
4. Create posts for blog content.
5. Create custom posts for tours using the 'tour' post type (available in admin menu).
6. Set up menus under Appearance > Menus.
7. Update content and images as needed in the WordPress editor.

## Page Templates

- `page-about.php`: About page with hardcoded content
- `page-contact.php`: Contact page with form and map
- `page-gallery.php`: Gallery page with images
- Other pages use `page.php` for standard layout
- Blog posts use `single.php`
- Home page uses `index.php`

## Features

- Custom post type for tours
- Responsive design
- All original assets included
- Same UI/UX as PHP site

## Notes

- The theme uses the same CSS, JS, and images from the original site.
- Content management is done through WordPress editor.
- No customizations needed in WordPress; just update content.
